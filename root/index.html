THE PAGE REPLACEMENT ALGORITHM DECIDES WHICH MEMORY PAGE IS TO BE REPLACED.

THE PROCESS OF REPLACEMENT IS SOMETIMES CALLED SWAP OUT OR WRITE TO DISK.PAGE REPLACEMENT IS DONE WHEN THE REQUESTED PAGE IS NOT FOUND IN THE MAIN MEMORY.THERE ARE 3 TYPES OF PAGE REPLACEMENT ALGORITHMS:
